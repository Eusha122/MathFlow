
## Installation

pip install numpy matplotlib

## Run

python function_all.py

## Usage

Enter a function using x (for normal graphs)
or use x and y (for shapes)

Type exit to quit

---

## Example Functions

### Normal Functions

sin(x)
cos(x)
x**2
x**3
abs(x)

---

### Advanced / Visual

exp(-0.1*x)*sin(5*x)
sin(x**2)
cos(x**2)
sin(x) + sin(2*x) + sin(3*x)

---

### Patterns

abs(sin(x))
abs(cos(x))
sin(x)*sin(x**2)
cos(x)*sin(x**2)

---

### Shapes (Implicit)

(x**2 + y**2 - 1)**3 - x**2*y**3
x**2 + y**2 - 4
abs(x) + abs(y) - 2
sin(x**2 + y**2) - 0.5

---

## Notes

Use ** for powers
Do not use ^
Implicit shapes must include both x and y
